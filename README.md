# Notes on Large Cardinals
This repository contains the entire LaTeX project of my personal notes on Large Cardinals, based on Paul Minter's and Benedikt Löwe's Lecture Notes, Jech's Book - Set Theory, and Kanamori's Book - The Higher Infinite.

## Pdf File
The compiled version of these notes is available [here](https://raw.githubusercontent.com/Xennonio/Large-Cardinals/main/Large_Cardinals.pdf).

## About the Notes
These notes are merely a comprehensive and filtered summary of Paul Minter's and Benedikt Löwe's Lecture Notes, Jech's Book: Set Theory and Kanamori's Book: The Higher Infinite, along with some of my own intuitions that may be useful to someone, and some additional comments on things I've found elsewhere.
